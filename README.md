# PantoConstruct (working title)

Just open the index.html in a browser, there is no setup required

This web app allows for easy construction of layouts for DualPanto Unity apps.

The generated .obj can be imported into Unity and used in conjunction with the new PantoMeshCollider (Also in this repo). The .obj asset needs to have the "Generate Colliders" box checked in Unity, and have a Unity MeshCollider attached to it, with the correct mesh selected.

Move the canvas around by holding shift. Drawing lines is done by clicking to set start and end points, drawing polygons is just done through clicking to create a vertex.
